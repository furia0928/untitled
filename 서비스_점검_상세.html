데이터 구성 항목 테이블

항목명 컨트롤 필수 여부 수정 가능 여부 설명 비고
제목 input(text) 필수 가능 서비스 점검 제목 입력 예: 11월 서비스 점검
시작일시 calendar + select 필수 가능 점검 시작 일시 선택 날짜 + 시간(시, 분)
종료일시 calendar + select 필수 가능 점검 종료 일시 선택 날짜 + 시간(시, 분)
점검 사유 textarea 필수 가능 점검 상세 사유 입력 최대 100자, 문자수 표시
상태 text(readonly) - 불가능 현재 점검 상태 표시 예정, 진행중, 완료, 취소됨
등록일시 text(readonly) - 불가능 등록일시 표시 YYYY-MM-DD HH:mm 형식
등록자 text(readonly) - 불가능 등록자명 표시 등록한 사용자명

처리 로직 정의

이벤트명 입력값/파라미터 처리내용 출력값/처리결과 비고
취소 버튼 클릭 없음 점검 취소 확인 팝업 표시 확인 다이얼로그 표시 예정 상태에서만 활성화
점검 취소 확인 사용자 확인 점검 상태를 취소됨으로 변경 점검 취소 처리 완료 복구 불가, 이력에만 남음
점검 취소 닫기 사용자 취소 확인 다이얼로그 닫기 상세 페이지로 복귀 취소 처리하지 않음
목록 버튼 클릭 없음 서비스 점검 관리 목록으로 이동 목록 페이지 표시 [Posting Guide] 적용
페이지 로드 점검 ID 해당 점검의 상세 정보 조회 상세 정보 표시 읽기 전용 필드 자동 설정
상태별 버튼 제어 점검 상태 상태에 따른 버튼 활성화/비활성화 취소 버튼 표시 여부 결정 예정 상태에서만 취소 가능
