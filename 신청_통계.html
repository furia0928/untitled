시작일자 datepicker 조건부 필수 가능 조회 시작 날짜 입력 기간 검색 탭 선택 시

서비스 카테고리 text - 불가 신청 서비스 종류 표시 스마트비즈엑스포, 판로지원 등
신청 상태 text - 불가 신청 처리 상태 표시 신청 접수, 승인, 반려 등

처리 로직 정의

이벤트명 입력값/파라미터 처리내용 출력값/처리결과 비고
기간 탭 클릭 탭 종류(일간/주간/월간) 선택한 기간 기준으로 데이터 조회 범위 설정 해당 기간의 통계 데이터 표시 일간: 오늘, 주간: 7일, 월간: 30일
검색 버튼 클릭 기간 정보 선택된 기간 내 신청 건수 통계 데이터 조회 바 차트 및 서비스별 현황 갱신 최대 1년까지 조회 가능
초기화 버튼 클릭 - 모든 입력값을 기본 상태로 리셋 일간 탭 선택, 오늘 날짜로 초기화 기본 상태로 복원
바 차트 표시 기간별 데이터 서비스별 신청 건수를 바 차트로 시각화 6개 서비스 신청 건수 차트 표시 스마트비즈엑스포, 판로지원, 기업 제품 홍보, 기업 제품 변경, 아리랑TV, 스마트365센터
서비스별 현황 표시 신청 데이터 각 서비스별 신청 상태와 건수를 카드 형태로 표시 서비스별 상세 현황 카드 리스트 전체 건수는 배경 색상으로 강조
기간 범위 표시 선택 기간 현재 조회 중인 기간 범위를 상단에 표시 "YYYY.MM.DD ~ YYYY.MM.DD" 형식 실시간 업데이트
상태별 필터링 신청 상태 각 서비스별로 신청 상태에 따른 건수 분류 신청 접수, 수정 접수, 승인, 반려, 수정 반려 건수 서비스에 따라 상태 종류 다름
전체 통계 계산 모든 신청 데이터 전체 신청 수를 모든 상태 포함하여 계산 서비스별 전체 합계 표시 신청 접수~수정 반려 모든 상태 포함
