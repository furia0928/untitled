데이터 구성 항목 테이블

항목명 컨트롤 필수 여부 수정 가능 여부 설명 비고
구분 select 필수 가능 약관의 종류 구분 드롭다운 선택
공개 radio(Y/N) 필수 가능 약관 공개 여부 설정 Y: 공개, N: 비공개
적용일자(버전) date 필수 가능 약관이 적용되는 날짜 입력 YYYY.MM.DD 형식
게시일자 date 필수 가능 약관이 게시되는 날짜 입력 YYYY.MM.DD 형식
내용 textarea 필수 가능 약관의 상세 내용 입력 리치 텍스트 에디터

처리 로직 정의

이벤트명 입력값/파라미터 처리내용 출력값/처리결과 비고
수정 버튼 클릭 폼 데이터 전체 입력값 유효성 검증 → DB 업데이트 → 성공/실패 처리 "수정되었습니다" 성공 메시지 또는 오류 메시지 필수 항목 누락 시 오류 표시
삭제 버튼 클릭 약관 ID 삭제 확인 팝업 → 사용자 확인 → DB에서 데이터 삭제 "삭제되었습니다" 메시지 또는 취소 처리 삭제 전 확인 팝업 필수
목록 버튼 클릭 - 현재 페이지에서 약관 목록 페이지로 이동 약관 관리 목록 페이지 이동 수정사항 미저장 시 확인 필요
날짜 선택 날짜 값 달력 팝업 표시 → 날짜 선택 → 입력 필드에 적용 YYYY.MM.DD 형식으로 입력 필드 업데이트 과거/미래 날짜 제한 가능
공개 여부 선택 Y 또는 N 라디오 버튼 상태 변경 → 선택값 저장 선택된 값으로 폼 상태 업데이트 기본값은 Y(공개)
