데이터 구성 항목 테이블

항목명 컨트롤 필수 여부 수정 가능 여부 설명 비고
구분 select 필수 가능 약관의 종류 구분 드롭다운 선택
공개 radio(Y/N) 필수 가능 약관 공개 여부 설정 Y: 공개, N: 비공개, 기본값 N 선택됨
적용일자(버전) date 필수 가능 약관이 적용되는 날짜 입력 YYYY.MM.DD 형식, 달력 아이콘
게시일자 date 필수 가능 약관이 게시되는 날짜 입력 YYYY.MM.DD 형식, 안내 메시지 포함
내용 textarea 필수 가능 약관의 상세 내용 입력 리치 텍스트 에디터, 다양한 포맷 지원

처리 로직 정의

이벤트명 입력값/파라미터 처리내용 출력값/처리결과 비고
저장 버튼 클릭 폼 데이터 전체 입력값 유효성 검증 → DB 업데이트 → 성공/실패 처리 "저장되었습니다" 성공 메시지 또는 오류 메시지 필수 항목 누락 시 오류 표시
취소 버튼 클릭 - 변경사항 확인 팝업 → 이전 페이지로 이동 약관 관리 목록 페이지 이동 또는 머무르기 수정사항 있을 경우 확인 팝업
공개 여부 선택 Y 또는 N 라디오 버튼 상태 변경 → 선택값 저장 선택된 값으로 폼 상태 업데이트 N이 기본 선택됨
날짜 선택 날짜 값 달력 팝업 표시 → 날짜 선택 → 입력 필드에 적용 YYYY.MM.DD 형식으로 입력 필드 업데이트 과거/미래 날짜 제한 가능
게시일자 정보 확인 - 게시일자 필드 포커스 시 안내 메시지 표시 "게시일 이전에는 약관이 노출되지 않습니다" 메시지 주의 아이콘과 함께 표시
내용 편집 텍스트 내용 리치 텍스트 에디터 기능 실행 → 포맷 적용 서식이 적용된 텍스트 내용 Bold, 밑줄, 목록, 링크 등 지원
