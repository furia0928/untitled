신청 프로그램 button(text) 선택 읽기전용 이전 참가 내역 조회 팝업 모달로 상세 내역 표시

처리 로직 정의

이벤트명 입력값/파라미터 처리내용 출력값/처리결과 비고
페이지 로드 입력값: 기업 ID 데이터베이스에서 해당 기업의 상세 정보 조회하여 화면에 표시 기업 상세 정보 폼 화면 화면 제목: "스마트공장 참여 기업 상세"
이전 참가 내역 조회 입력값: 기업 ID 스마트공장 참여 기업 인증 팝업 노출 → 정상 인증 완료 시 이전 참가 내역 모달 팝업 표시 "신청 기업 이전 참가 내역" 모달 팝업 (전체 11건 등) 인증 여부에 따라 접근 가능 여부 분기
수정 버튼 클릭 입력값: 현재 폼 데이터 유효성 검사 수행 → 필수 항목 체크 → 데이터베이스 업데이트 수정 성공/실패 메시지 및 목록 페이지로 이동 수정일시, 수정자 자동 업데이트
삭제 버튼 클릭 입력값: 기업 ID 삭제 확인 팝업 표시 → 확인 시 해당 기업 데이터 삭제 "정말 삭제하시겠습니까?" 확인 팝업 및 삭제 결과 삭제 후 목록 페이지로 자동 이동
목록 버튼 클릭 입력값: 없음 현재 페이지를 종료하고 기업 목록 페이지로 이동 기업 목록 페이지 화면 수정된 내용이 있을 경우 저장 확인 팝업
참여내역 연도 선택 입력값: 연도 (YYYY) 선택한 연도에 해당하는 프로그램 목록 필터링 해당 연도 프로그램 목록 및 입력 필드 활성화 연도별 프로그램 데이터 동적 로딩
메모 입력 입력값: 텍스트 (최대 100자) 실시간 글자 수 체크 → 100자 초과 시 입력 제한 글자 수 카운터 표시 (예: 0/100) 실시간 유효성 검사 적용
모달 팝업 닫기 입력값: 없음 이전 참가 내역 모달 팝업 닫기 메인 폼 화면으로 복귀 X 버튼 또는 ESC 키로 닫기 가능
폼 유효성 검사 입력값: 전체 폼 데이터 필수 항목 체크 → 사업자등록번호 형식 검증 → 기업명 중복 체크 유효성 검사 결과 및 오류 메시지 실시간 검증 또는 제출 시 검증
