등록 버튼 클릭 - 관리자 등록 페이지로 이동 페이지 이동 (BADM01010300) 신규 관리자 등록 페이지 호출
이름 클릭 선택된 관리자 정보 해당 관리자 상세(수정) 페이지로 이동 페이지 이동 (BADM01010200) 관리자 정보 수정 가능
페이지네이션 클릭 페이지 번호 해당 페이지의 관리자 리스트 조회 선택된 페이지의 데이터 표시 테이블 하단 중앙에 위치
페이지 로드 - 관리자 리스트 조회 (등록일 기준 내림차순 정렬) 관리자 목록 표시 및 전체 건수 표시 (전체 N,NNN건) 최신 등록 순으로 정렬
